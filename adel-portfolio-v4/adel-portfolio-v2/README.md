# Adel Sinaga — Personal Portfolio

A premium, multi-page personal portfolio website for Adel Septiana Sinaga, an Information Technology student at President University.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Project Structure

```
app/
├── page.tsx                          # Home (with Loading Screen)
├── about/page.tsx                    # About Me
├── skills/page.tsx                   # Skills
├── projects/page.tsx                 # Projects Listing
├── projects/property-rental/page.tsx
├── projects/eventkita/page.tsx
├── projects/student-budgeting/page.tsx
├── projects/before-satan-catch-you/page.tsx
├── experience/page.tsx               # Experience Timeline
├── contact/page.tsx                  # Contact Form
└── globals.css

components/
├── Navbar.tsx                        # Responsive Navigation
├── LoadingScreen.tsx                 # Animated Intro Screen
└── ProjectLayout.tsx                 # Reusable Project Page Layout
```

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Features

- ✅ Loading Screen with typing animation & particle background
- ✅ Animated Home with mouse-tracking glow & floating icons
- ✅ About page with animated counters
- ✅ Skills organized in 4 categories with hover effects
- ✅ Projects card grid with individual case-study pages
- ✅ Animated timeline Experience page
- ✅ Contact form with social links & availability status
- ✅ Glassmorphism UI throughout
- ✅ Fully responsive (mobile → desktop)
- ✅ Framer Motion animations on every page
